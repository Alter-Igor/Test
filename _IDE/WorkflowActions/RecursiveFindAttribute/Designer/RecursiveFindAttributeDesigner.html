Use this action to find an attribute on a work item and if required keep searching though its parents until the value is found or to a maximun depth as specified below

NOTE: This action will only search through the parents of the work item if the attribute is not found on the work item itself.

This is the name of the attribute to search for. This could be a custom attribute or a standard attribute such as "Title" or "Description."

In the case of a custom attribute, the path to the attribute can be specified using the dot notation, e.g., "formbuilderName.fieldName".

For an option set on a form builder field, the path to the option set can be specified using the dot notation, e.g., "formbuilderName.fieldName.meaningCode".
